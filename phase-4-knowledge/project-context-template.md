# 项目上下文文档（Project Context）

> **定位：** 本文档参考 BMad 的 project-context.md 概念，是项目知识的"快速恢复"载体。当沙盒重置、会话切换或 AI 模型更换时，加载本文档即可快速恢复项目的完整上下文，避免从零开始。
>
> **核心理念：** project-context.md 是项目知识的"压缩包"——它不包含所有细节，但包含了恢复完整上下文所需的所有关键信息。新会话中的 AI 加载本文档后，应能理解项目的全貌并继续工作。
