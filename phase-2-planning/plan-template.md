每个任务必须遵循以下规范（参考 Superpowers Writing Plans 技能）：

1. **每步 2-5 分钟**：每个任务应在 AI 执行时耗时 2-5 分钟
2. **一个动作**：每个任务只包含一个明确的操作
3. **无占位符**：禁止使用 TODO、待定、适当、合理等模糊表述
4. **可验证**：每个任务有具体的测试命令和预期输出

## 执行约定

### 执行顺序

1. 按任务编号顺序执行，严格遵守依赖关系
2. 每完成一个任务，立即运行对应的测试命令验证
3. 如果测试失败，立即修复，不得跳过或标记为 TODO
4. 每完成一个阶段，运行阶段性验收标准验证

### 变更管理

- 如果在执行过程中需要修改计划，必须：
  1. 记录变更原因
  2. 评估变更影响（涉及哪些后续任务）
  3. 更新任务列表和依赖关系
  4. 通知使用者确认（如影响范围较大）

### 阻塞处理

- 如果某个任务因外部原因被阻塞（如需要使用者提供信息）：
  1. 标记任务状态为"阻塞"
  2. 记录阻塞原因
  3. 评估是否可以跳过该任务继续执行后续任务
  4. 如果不可跳过，暂停执行并通知使用者
